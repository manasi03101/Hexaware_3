* Git

programmers collaboratively developing source code during software development

*GitHub

*Git Command
1. git config
-> This command sets the author name and email address respectively to be used with your commits

2.git init
-> used to start a new repository.

3. git clone
-> It is used to obtain a repository from an existing URL.

4.git add
-> This command adds a file

5. git commit
->This command records the file permanently in the version history

6. git status
-> lists all the files that have to be committed.

7.git branch
-> Used to create a new branch

8.git push
-> This command sends the branch commits to your remote repository.

9. git pull
->This command fetches and merges changes on the remote server to your working directory.
